# Anti-Fraud Neural Shield

This skill focuses on 'Financial Security', identifying behavioral anomalies in payment flows and communication to block bad actors.

## Instructions
1. Analyze historical transaction patterns to build a 'Clean' baseline.
2. Flag any deviation (Ex. Large withdrawal from a new IP) for multi-factor verification.
3. Use NLP to scan incoming messages for phishing signatures.
4. Automatically rotate session keys if a threat is detected.

## Examples
- "Audit our latest 100 payments for potential wash-trading patterns."
