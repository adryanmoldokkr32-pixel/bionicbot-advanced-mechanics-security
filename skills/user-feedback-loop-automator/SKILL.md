# User Feedback Feedback-Loop

This skill focuses on the 'Product-Market Fit', turning user complaints into product improvements automatically.

## Instructions

1. Scrape feedback from Discord, Reddit, App Store, or Support tickets.
2. Use sentiment analysis to group feedback into 'Pains' and 'Desires'.
3. Match feedback to existing features or backlog items.
4. Prioritize tasks using the RICE method (Reach, Impact, Confidence, Effort).
5. Notify users automatically when their requested feature is launched.

## Examples

- "Summarize the top 3 user complaints from the last 7 days of Discord logs."
- "Turn this user email into a prioritized ticket in our GitHub backlog."
