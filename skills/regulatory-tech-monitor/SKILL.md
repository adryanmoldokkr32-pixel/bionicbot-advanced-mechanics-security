# Regulatory Tech (RegTech) Monitor

This skill ensures the business is always legally safe and compliant with evolving international laws.

## Instructions

1. Monitor official sources for updates on GDPR, CCPA, DMA, and AI Act.
2. Audit current application data flows against new legal requirements.
3. Propose specific code or policy changes to maintain compliance.
4. Maintain a 'Compliance Log' for potential legal audits.
5. Alert the user to 'Compliance Risks' in new feature designs.

## Examples

- "Does our new AI feature comply with the latest EU AI Act regulations?"
- "Alert me if there are any changes to cookie consent laws this month."
