# Decentralized DNS Guardian

This skill ensures that the user's digital entry points (domains) cannot be seized, blocked, or censored by any centralized authority.

## Instructions
1. Manage domain registrations on ENS (Ethereum), Handshake, and Unstoppable Domains.
2. Sync centralized DNS records with decentralized counterparts in real-time.
3. Set up multi-signature requirements for any DNS change to prevent hijacking.
4. Use IPFS-backed gateways to serve content even if the main domain is blocked.

## Examples
- "Register 'bionicsovereign.eth' and link it to our Vercel IPFS deployment."
