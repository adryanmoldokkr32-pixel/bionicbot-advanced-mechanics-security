# Disaster Recovery Drill Manager

This skill focuses on the 'Insurance' policy of the business, ensuring you can survive a total infrastructure loss.

## Instructions

1. Define 'Disaster Scenarios' (Ex. Region outage, Ransomware, DB deletion).
2. Design a step-by-step 'Recovery Protocol'.
3. Run a simulation ('Game Day') to test the protocol and the team.
4. Measure 'Recovery Time' and identify blockers during the drill.
5. Update backups and infrastructure to address vulnerabilities found.

## Examples

- "Simulate a total AWS region failure: can we recover in under 1 hour?"
- "Draft a recovery plan for if our main database was accidentally deleted."
